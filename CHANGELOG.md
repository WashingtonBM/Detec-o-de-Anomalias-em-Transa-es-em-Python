# 📝 Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2026-05-20

### 🎉 Adicionado
- ✨ Versão inicial do projeto
- 🤖 Modelo XGBoost treinado com 100% de acurácia
- 📊 Análise SHAP para explicabilidade
- 🛡️ Detecção de fraudes com 85% de recall
- 📈 Visualizações (Matriz de Confusão, Feature Importance, SHAP)
- 💾 Exportação de modelo via joblib
- 📚 Documentação completa (README)
- 📦 requirements.txt com dependências
- 🐳 Docker e docker-compose para containerização
- 🌐 Exemplo de API FastAPI para produção
- 📄 Licença MIT
- 🤝 Guia de contribuição (CONTRIBUTING.md)
- .gitignore configurado
- Model cards e informações do projeto

### 📝 Documentação
- README.md com guia completo de instalação e uso
- Exemplos de código para predições
- Explicação dos hiperparâmetros
- Guia de deploy em produção

### 🔧 Configuração
- Python 3.8+ suportado
- Testes em XGBoost 2.0+, scikit-learn 1.3+, pandas 2.0+
